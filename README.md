# angular6-boot2
Sample CRUD app with Angular 6 and Spring Boot 2

Required software:
For Angular 6:
- node.js 8.x or greater
- npm 5.x or greater
- ng 6.x or greater

For Spring Boot 2:
- JDK 8.X or greater
- Maven

sample-app is an Angular project.
RestDemo is an Spring Boot project.


To run angular project:
npm istall
ng serve --open

To run boot project:
mvn spring-boot:run
